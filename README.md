📸 ShutterStore

A backend REST API for an e-commerce camera store built with Java and Spring Boot.

🚀 Features
User Registration & Login
JWT Authentication
Role-Based Authorization (USER / ADMIN)
Email Verification
Product, Category & Brand Management
Cart Management
Order Management
Reviews & Wishlist
Request Validation
Global Exception Handling
Swagger API Documentation
🛠️ Tech Stack
Java 17
Spring Boot 3
Spring Security
Spring Data JPA
Hibernate
JWT
Spring Mail
Bean Validation
MapStruct
Lombok
Maven
SQL Server
Swagger (OpenAPI)
